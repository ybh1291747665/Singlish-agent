# Public Examples

This folder is for short, redistributable sample assets that are safe to publish with the repository.

Intended contents:

- Small public demo audio clips
- Expected outputs for those clips
- Screenshot-ready examples for README or demo flows

Current state:

- No public sample audio has been committed yet.

When adding files here:

- Keep them short and clearly licensed.
- Document the source or recording permission.
- Add a matching expected result file when possible, such as:
  - `example-001.expected.md`
  - `example-001.expected.srt`
  - `example-001.expected.json`
